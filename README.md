# 신장이식환자 건강관리 PWA

KDIGO 가이드라인 기반 신장이식 환자 일상 점검 앱

**앱 주소**: https://kalparho.github.io/kidney-health/

---

## 주요 기능

### 오늘 체크 탭
- 약물 복용 체크리스트 (오전·오후·저녁 시간대별)
- 건강 측정 기록 (혈압, 혈당, 체중, 체온, 소변량)
- 생활습관 점검 (수분 섭취, 운동, 수면, 금연·금주)
- 식이 관리 (권장 식품 / 주의 식품 태그)
- 진료 예약 D-day 배너 (다음 진료일 알림)
- 이식 후 경과일 표시

### 캘린더 탭
- 월별 달성률 시각화 (완료 / 부분 / 미완료 점 표시)
- 진료 예약일 별도 마커 표시
- 날짜 클릭 시 상세 기록 및 예약 정보 조회

### 통계 탭
- 주간·월간 달성률 차트
- 건강 측정값 추이 그래프 (혈압, 체중 등)

### 설정 탭
- 프로필 설정 (이름, 이식일)
- 약물 복용 시간·이름 커스터마이징
- 진료 예약 관리 (병원명, 종류, 날짜 등록/삭제)
- 데이터 내보내기 / 가져오기 (JSON)

---

## 기술 스택

| 항목 | 내용 |
|------|------|
| 구현 방식 | 단일 HTML 파일 PWA |
| 데이터 저장 | LocalStorage (`kt_` 접두사) |
| 오프라인 지원 | Service Worker (Cache-First) |
| 설치 | iOS: Safari → 공유 → 홈 화면에 추가 |
| 기반 가이드라인 | KDIGO / NKF / AST / 대한신장학회 |

---

## 파일 구조

```
output/
├── index.html          # 앱 메인 (HTML + CSS + JS 통합)
├── manifest.json       # PWA 설정
├── service-worker.js   # 오프라인 캐시
└── icon.svg            # 앱 아이콘
```

---

## 설치 방법 (아이폰)

1. Safari에서 https://kalparho.github.io/kidney-health/ 접속
2. 하단 공유 버튼 탭
3. **홈 화면에 추가** 선택
4. 앱 이름 확인 후 **추가**
